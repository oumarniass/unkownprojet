How to activation PHPStorm 2017 — 2018 License Server

root    06.08.2017 24.07.2018    98 комментариев к записи How to activation PHPStorm 2017 — 2018 License Server

В интернете достаточно много инструкций как активировать продукты от JetBrains, в том числе и PHPStorm. Но большая часть статей уже устаревшая, и указанные сервера для активации продуктов уже не доступны. Здесь я буду дополнять и постоянно обновлять сервера, для активации приложения PHPStorm.
Способ достаточно простой, и безопасный. Для параноиков так же имеется, версия для локальной активации PHPStorm.

Активация PHPStorm 2017.3 and 2018 с помощью интернет серверов:

  1. http://66.42.52.162:8000
  2. http://www.activejetbrains.gq/ (2018.1.x)
  3. http://nodde.ru:1027
  4. http://80.211.190.88:1027
  5. http://jetbrains.tools
  6. https://idea.qmanga.com/
  7. http://jetbrains.license.laucyun.com/
  8. http://144.202.4.96 (PhpStorm 2018.1.3)
  9. http://80.211.190.88:1027

Activation locale (sans accès Internet):

  • Téléchargez le fichier LocalServer.zip sur votre ordinateur (mot de passe pour l'archive: 1234).
  • Décompressez l'archive et exécutez le fichier correspondant au nom de votre système d'exploitation.
  • Exécutez ReSharper et ajoutez notre adresse d’activation locale dans l’onglet Serveur de licences http://127.0.0.1:1017/.
  • Cliquez sur OK
